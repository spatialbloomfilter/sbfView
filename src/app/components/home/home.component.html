Spatial Bloom Filter

The Spatial Bloom Filters (SBF) are a compact, set-based data structure that extends the original Bloom filter concept. An SBF represents and arbitrary number of sets, and their respective elements (as opposed to Bloom filters, which represent the elements of a single set). SBFs are particularly suited to be used in privacy-preserving protocols, as set-membership queries (i.e. the process of verifying if an element is in a set) can be easily computed over an encrypted SBF, through (somewhat) homomorphic encryption. Similarly to a classical BF, a SBF is a space-efficient probabilistic data structures used to test whether an element is a member of a set. SBF improves this feature allowing the use of more than one set. That is, with the SBF it is possible identify the specific set of an element over multiple sets (possibly an unlimited number). The SBF retains all the properties of teh BF like

  • constant time for write and read of a filter's element
  • memory usage optimization
  • usage of metrics to setup the filter (false positive rate)

Moreover the SBF introduce two more metrics: Emersion and Safeness. The SBF is specifically designed to deal with location information to keep private. However this data structure is suitable for any kind of set-based problem.
